# PMU Academic Hiring Management System

A web-based academic hiring management system developed for Prince Mohammad Bin Fahd University (PMU). The system digitizes and streamlines the faculty hiring process across multiple organizational roles, from requisition creation through final appointment decisions.

---

## 👥 System Roles

| Role | Responsibilities |
|---|---|
| **Dean** | Reviews shortlists, makes final hiring decisions, manages requisitions |
| **HR** | Creates and manages job requisitions, oversees candidate pipeline |
| **Department Chair** | Reviews candidates, submits recommendations, approves interview schedules |
| **Interview Committee** | Evaluates candidates, submits structured evaluation forms |

---

## ⚙️ Core Features

- **Candidate Management** — Track applicants through each stage of the hiring pipeline
- **Interview Scheduling** — Committee members schedule and manage interviews with chair approval
- **Evaluation Workflow** — Structured evaluation forms submitted per candidate per interviewer
- **Shortlist Builder** — Chairs compile and submit ranked shortlists for dean review
- **Dean Final Decisions** — Dean reviews all evaluations and recommendations before approving hires
- **Gemini AI Analysis** — AI-assisted candidate analysis to support hiring decisions
